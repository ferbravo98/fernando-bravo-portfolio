Consultorio Médico

Aplicación de escritorio orientada a la gestión de pacientes, consultas, historia clínica, estudios, vacunación y generación de documentos. Es mi proyecto principal por alcance, utilidad real y evolución técnica.

Stack usado

Java · JavaFX · SQL · Git · PDF

Resumen

¿Qué problema resuelve?

El sistema busca digitalizar y ordenar información médica que normalmente puede quedar dispersa en registros manuales, planillas o documentos separados.

Funcionalidades principales

  • Registro y administración de pacientes.
  • Búsqueda y actualización de datos del paciente.
  • Gestión de consultas e historia clínica.
  • Registro de estudios y antecedentes relevantes.
  • Control de vacunación.
  • Generación de documentos PDF.

Enfoque técnico

El proyecto fue desarrollado como una aplicación de escritorio con Java y JavaFX, separando la interfaz, la lógica de negocio y el acceso a datos. La persistencia permite conservar información clínica y consultarla de forma ordenada.

Valor técnico

Qué demuestra este proyecto

Más allá de la interfaz, este proyecto muestra lógica de negocio, persistencia, modelado de datos y construcción de una herramienta útil para un usuario real.

Habilidades aplicadas

  • Diseño de una aplicación con múltiples módulos.
  • Manejo de formularios, tablas y navegación entre vistas.
  • Persistencia de datos mediante base SQL.
  • Validación y actualización de información.
  • Generación de reportes o documentos exportables.

Aprendizajes

Este proyecto me permitió acercarme a un flujo real de trabajo: entender una necesidad concreta, convertirla en funcionalidades, ordenar la información y construir una herramienta mantenible para mejorar un proceso cotidiano.
